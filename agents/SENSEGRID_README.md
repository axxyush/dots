## SenseGrid (Agentverse)

SenseGrid is a chat agent that turns a floorplan image URL into:
- a reconstructed PNG preview
- structured JSON
- a tactile map image (Nano Banana Pro) for accessibility workflows

### How to use

1. Send a **public http(s) URL** to a floorplan image (PNG/JPG).
2. The agent will quote a price and ask you to reply **`yes`**.
3. You’ll get a blurred preview, then final links once the flow completes.

### Notes
- Runs locally but is reachable via Agentverse mailbox relay (`mailbox=True`).
